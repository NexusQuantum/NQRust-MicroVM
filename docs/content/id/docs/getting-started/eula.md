Saat login pertama kali, setiap pengguna harus membaca dan menyetujui Perjanjian Lisensi Pengguna Akhir (EULA) sebelum mengakses NQRust-MicroVM. Layar EULA muncul secara otomatis dan tidak dapat dilewati.

{{< img src="/images/license/eula.png" alt="Layar End User License Agreement" >}}

---

## Menyetujui EULA

1. Baca teks perjanjian lengkap di layar.
2. Centang **"I have read and agree to the End User License Agreement"**.
3. Klik **I Accept**.

Anda hanya perlu menyetujui sekali per akun pengguna. Setelah diterima, layar EULA tidak akan muncul lagi untuk akun tersebut.

{{% alert icon="ℹ️" context="info" %}}
EULA memiliki versi. Jika Nexus Quantum Tech menerbitkan versi baru perjanjian, semua pengguna akan diminta untuk menyetujuinya saat login berikutnya.
{{% /alert %}}

---

## Ringkasan EULA

Teks hukum lengkap ditampilkan di UI. Poin-poin utama:

| Pasal | Ringkasan |
|---|---|
| **Hak Kekayaan Intelektual** | Semua hak cipta, paten, kode sumber, dan model AI adalah milik eksklusif Nexus Quantum Tech. |
| **Pemberian Lisensi** | Anda menerima hak penggunaan yang terbatas, tidak eksklusif, dan tidak dapat dipindahtangankan. Ini bukan penjualan produk. |
| **Batasan Instalasi** | Instalasi dan aktivasi dibatasi sesuai jumlah lisensi yang dibeli. |
| **Penggunaan Terlarang** | Rekayasa balik, distribusi ulang, dan sublisensing dilarang. |
| **Penghentian** | Lisensi berakhir secara otomatis jika ada pelanggaran terhadap ketentuan. |

{{% alert icon="⚠️" context="warning" %}}
Dokumentasi ini hanya ringkasan. Teks resmi adalah perjanjian yang ditampilkan di UI. Jika Anda tidak setuju dengan ketentuan, jangan gunakan perangkat lunak ini.
{{% /alert %}}

## Bahasa

EULA tersedia dalam **Bahasa Inggris** dan **Bahasa Indonesia**. Gunakan pemilih bahasa (ikon globe) di pojok kanan atas layar EULA untuk beralih bahasa sebelum membaca dan menyetujui.
