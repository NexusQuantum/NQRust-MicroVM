# Memulai

NQRust-MicroVM dilengkapi dengan installer TUI berbasis Rust yang menangani segalanya di host Linux baru — dependensi, pengaturan KVM, jaringan, database, layanan systemd, dan konfigurasi awal.

---

## Yang Anda Butuhkan

**Perangkat Keras**
- CPU x86_64 dengan dukungan KVM (Intel VT-x atau AMD-V)
- RAM minimal 4 GB, direkomendasikan 8 GB
- 20 GB ruang disk kosong

**Perangkat Lunak**
- Linux — direkomendasikan Ubuntu 22.04+ atau Debian 11+
- Akses `sudo`

---

## Pilihan Instalasi

### [Panduan Instalasi](installation/)
Siapkan NQRust-MicroVM menggunakan installer **online** atau **airgapped**.

**Waktu yang dibutuhkan:** ~10 menit

### [Quick Start](quick-start/)
Masuk ke platform yang baru diinstal dan buat microVM pertama Anda.

**Waktu yang dibutuhkan:** ~5 menit (setelah instalasi)

### [Aktivasi Lisensi](license/)
Aktifkan NQRust-MicroVM menggunakan kunci lisensi online atau file `.lic` offline.

### [EULA](eula/)
Baca dan setujui Perjanjian Lisensi Pengguna Akhir yang diperlukan saat login pertama kali.
