# Getting Started

NQRust-MicroVM ships with a Rust-based TUI installer that handles everything on a fresh Linux host — dependencies, KVM setup, networking, database, systemd services, and initial configuration.

---

## What You'll Need

**Hardware**
- x86_64 CPU with KVM support (Intel VT-x or AMD-V)
- 4 GB RAM minimum, 8 GB recommended
- 20 GB free disk space

**Software**
- Linux — Ubuntu 22.04+ or Debian 11+ recommended
- `sudo` access

---

## Installation Options

### [Installation Guide](installation/)
Set up NQRust-MicroVM using the **online** or **airgapped** installer.

**Time required:** ~10 minutes

### [Quick Start](quick-start/)
Log in to your freshly installed platform and create your first microVM.

**Time required:** ~5 minutes (after installation)
