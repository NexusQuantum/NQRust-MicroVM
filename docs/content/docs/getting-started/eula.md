On first login, every user must read and accept the End User License Agreement (EULA) before accessing NQRust-MicroVM. The EULA screen appears automatically and cannot be skipped.

{{< img src="/images/license/eula.png" alt="End User License Agreement screen" >}}

---

## Accepting the EULA

1. Read the full agreement text on screen.
2. Tick **"I have read and agree to the End User License Agreement"**.
3. Click **I Accept**.

You will only need to accept once per user account. After acceptance the EULA screen will not appear again for that account.

{{% alert icon="ℹ️" context="info" %}}
The EULA is versioned. If Nexus Quantum Tech publishes a new version of the agreement, all users will be prompted to accept it on next login.
{{% /alert %}}

| **License Grant** | You receive a limited, non-exclusive, non-transferable right to use the software. This is not a sale. |
| **Installation Restrictions** | Installation and activation are restricted to the number of licenses purchased. |
| **Prohibited Use** | Reverse engineering, redistribution, and sublicensing are prohibited. |
| **Termination** | The license terminates automatically upon breach of any term. |

{{% alert icon="⚠️" context="warning" %}}
This documentation is a summary only. The authoritative text is the agreement shown in the UI. If you do not agree to the terms, do not use the software.
{{% /alert %}}

## Language

The EULA is available in **English** and **Bahasa Indonesia**. Use the language selector (globe icon) in the top-right of the EULA screen to switch languages before reading and accepting.
